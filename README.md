# Echoes of a Poetic Age

这是根据概念图制作的本地档案馆网站。可直接双击 `index.html` 阅读；需要把书籍草稿正式写入项目时，请双击 `启动档案馆.cmd`。

## 页面结构

- 首页：世界观主视觉与进入档案馆按钮
- 吟游诗人的见闻：十二本书的书架
- 阅读页：书封、章节目录、正文、前后章切换
- 写在前面的话、地图、人物、美术馆、设置：已完成基础页面与交互

## 本地导入 Word 内容

导入、预览和编辑全部在浏览器本地完成，不调用 AI，也不消耗 tokens。

1. 双击 `启动档案馆.cmd`
2. 在“设置”中开启“开发者模式”
3. 前往“吟游诗人的见闻”，在本地书籍工作台选择 `.docx`
4. 检查分卷与章节映射，应用为本地草稿
5. 预览确认后，点击“保存到项目文件”

Word 默认使用以下层级：

- 一级标题：书卷
- 二级标题：章节
- 三级至六级标题：章内标题
- `Subtitle` / “副标题”样式：书卷简介或章节副标题

导入器支持段落、标题等级、粗体、斜体、下划线、引用、列表、对齐、字体与相对字号。表格、正文图片、脚注、页眉页脚和批注会被忽略并在预览中提示。

直接双击 `index.html` 时仍可导入和本地编辑，但浏览器安全限制不允许写入项目文件。通过 `启动档案馆.cmd` 打开后，“保存到项目文件”才会启用。

正式保存时会：

- 校验 `books.js` 是否被其他程序修改
- 备份原文件到 `backups/`
- 将原始 Word 归档到 `source-documents/`
- 将新书封写入 `assets/covers/`
- 安全更新 `books.js`

## 直接编辑章节

开启开发者模式后，阅读页顶部会显示“编辑本章”。修改会在约 800ms 后自动保存到浏览器本地。工具栏支持正文、章内标题、引用、列表、粗体、斜体、下划线与对齐。

本地草稿不会自动改写项目。回到书籍工作台点击“保存到项目文件”后才会正式发布；也可以单独将某一卷恢复为项目版本。

## 书籍数据

十二卷正式数据保存在 `books.js`。新导入章节使用结构化 `blocks` 保存富文本，同时继续兼容旧版 `paragraphs` 数据。

## 添加人物

人物图鉴默认保持为空，不会自动读取文档或预置人物。

在网站“设置”里打开“开发者模式”，然后回到“人物图鉴”页面，即可上传图片、填写姓名与简介。开发者模式下还可以修改或删除已有的人物卡。

人物数据和图片保存在当前浏览器本地。

需要把浏览器本地人物正式写入项目时：

1. 在人物页开发者面板点击“导出人物数据”
2. 将下载的 `echoes-characters-日期.json` 交给 Codex
3. Codex 会使用 `tools/import-characters-export.js` 把图片写入 `assets/characters/`，并把人物信息写入 `characters.js`

正式写入项目后，开发者模式仍可以修改或删除人物。此类变更会先保存在浏览器本地；需要再次正式写入项目时，重新导出人物数据并导入即可。

## 添加后续地图

地图图册统一保存在 `maps.js`。

新增地图时：

1. 把地图图片放进 `assets/maps/`
2. 在 `maps.js` 的 `window.ATLAS_MAPS` 里追加一项

```js
{
  id: "new-map",
  title: "地图中文名",
  english: "English Name",
  image: "assets/maps/new-map.jpg",
  note: "简短说明"
}
```

## 管理美术馆

美术馆正式数据保存在 `gallery.js`，图片分别保存在项目根目录的 `编年史/`、`土地/` 与 `神话/` 文件夹。

需要添加或整理画作时：

1. 双击 `启动档案馆.cmd`
2. 在“设置”中开启“开发者模式”
3. 前往“美术馆”，选择分类并上传图片
4. 可在画作卡片下方改名、移动分类、删除，或拖动调整当前分类顺序
5. 检查草稿后点击“保存到项目”

启动脚本会让保存服务在后台持续运行，启动完成后命令窗口会自动关闭。若页面提示尚未连接，可以点击“重新检测连接”。

画作显示名称与真实图片文件名保持一致。保存时会统一校验并备份受影响的文件；直接双击 `index.html` 时仍可浏览，但不能写入项目。

## 视觉素材

- `assets/page-background.png`：全站页面统一使用的旅行者与雪山背景
- `assets/hero-landscape.png`：使用概念图作为风格参考生成的无文字世界景观
- `assets/covers/`：从书架概念图整理出的十二本独立封面
- `assets/characters/`：预留的人物图鉴图片目录
- `assets/maps/`：地图图册图片
- `assets/music/`：设置页可切换的背景音乐

## 文字防复制

公开页面默认禁止选择文字、复制、剪切、相关快捷键与右键菜单。搜索框、表单字段、下拉框以及开发者模式下正在编辑的章节不受限制，可以正常复制、剪切和粘贴。

此功能用于阻止浏览器中的常规复制操作，无法阻止开发者工具、查看静态源文件、截图或直接读取项目数据。
